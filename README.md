# @domlog/react-log

A tiny react component for logging!

## Test

```sh
npm test
```

## Pack

```sh
npm pack
```

## Linking locally, from project directory

```sh
npm link @domlog/react-log
```

and import as

```js
import { Log } from "@domlog/react-log";
```

## Usage on other projects

```sh
npm link @debug/react
```
